The Process

How I go about a project is the following:

  • Analyze what needs to be done
  • Design the process needed
  • Build the application and/or website
  • Run Tests for UI / UX
  • Run additional tests for potential bugs
  • Release results for viewing
  • Receive user feedback
  • Build customer rapport
  • Re-deliver with mobile responsiveness
  • Develop databases and /or other backend frameworks
  • Test I/O Channels
  • Final UI/UX testing
  • Devliver final product
  • Test real-life user compliance.
  • Update application where bugs are necessary
